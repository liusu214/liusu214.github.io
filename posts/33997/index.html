列表

列表是可变类型，用[]或list来定义

  • 列表操作
    list[i:j:k] 列表从0 <= n < (j-i)/k 的索引号 x = i + n* k 的切片
  • list[i:-1]不包括最后一个元素，其他同理*
    list.index(a) a在列表中首次出现的索引
    len(list)、max(list)、min(list) 列表的长度、最大值、最小值
    del list 删除列表
    list[index] = x 将列表index项替换为i,找不到会报错
    list.append()在列表末尾添加元素
    list.insert(index,a)在特定位置插入元素，耗费计算量大，因为对后续元素的引用必须在内部迁移，以便为新元素提供空间。
  • 当index是-1时，会插入到最后一个值前*
    list.pop(index)删除指定位置的元素，并返回该元素。不填默认删除最后一个，找不到会报错
    list.remove(值)删除第一个值，找不到会报错
    list.extend()可以添加多个元素
    list.sort()可以对列表进行排序，可以添加key指定排序规则，可以设置reverse=True进行反向排序(默认为False)
    list.reverse()反转列表
    ‘-‘.join(list)，以-为串联符号将列表内元素串联成字符串
  • 序列函数
    enumerate函数可以返回(index,value)的元组序列
    sorted函数，跟sort类似
    zip函数,可以将多个列表、元组或其他序列成对组合成一个元组列表

zip可以被用来解压序列，也可以转换行列表为列列表

reversed(list) 反转序列，返回一个迭代器

字典

字典是可变类型，用dict()来定义，以键值对的形式表示

  • 字符串操作
    dict[‘key’]访问与键对应的值，没有会报错
    dict[key] = value，若有则覆盖，没有则新建
    del dict 删除字典 dict.pop[key] 删除键值对，没有会报错 dict.popitem()从字典中移除最后一个并返回一个键值对
    dict.keys() dict.value() 输出键和值
    dict.update(dict1) 将一个字典和另一个融合
    dict.get(key) 返回值，*不会报错 *

dict.setdefault(键,数据) 键值对不存在添加键值对，存在则不做处理
dict.setdefault(key, []).append(value) 用于for循环中没有key就新建列表，有则添加到列表中
dict.items()获取所有键和值，用两个遍历变量
dict.values() 返回由字典值组成的新视图

  • 字典方法
    len(dict) 返回字典的项数
    iter(dict) 返回以字典的键为元素的迭代器

字符串

字符串为不可变类型，用str()转化为字符串

  • 字符串操作
    string.isalpha()
    所有都是字母返回True
    string.isdecimal()
    所有都是数字返回True
    string.islower()
    都是小写字母返回True
    string.isupper()
    都是大写字母返回True
    string.startswith(“a”)
    检查字符串是否是以 a开头，是则返回 True
    string.endswith(“a”)
    检查字符串是否是以 a结束，是则返回 True
    string.find(str, 0,5) 查找数据第一次出现的索引值，找不到返回-1，也可以指定范围查找
    string.rfind(str) 反向查找
    string.count(数据) 类似find()，找不到会报错
    string.index(str) 类似rfind()，找不到会报错
    string.replace(‘旧内容’,’新内容’, ‘替换次数’) 替换字符串中的内容，实际上是新建一个字符串
    string.split(str, maxsplit=-1) 以str为分隔符拆分成列表并返回，给出maxsplit则进行maxsplit此拆分
    string.lower()转为小写
    string.upper()转为大写
    string.lstrip() 把左边的空格去除
    string.rstrip()把右边的空格去除
    string.strip() 把左右两边的空格去除
    str.capitalize() 返回原字符串的副本，首个字符大写，其他小写
    str.casefold() 返回原字符串消除大小写的副本
    str.center(width,fillchar) 返回长度为width的字符串，原字符串在中间用fillchar填充两边的空位， 若width小于等于len(str)则返回原字符串的副本
    str.rjust(width[, fillchar])
    返回长度为 width 的字符串，原字符串在其中靠右对齐,
    使用指定的 fillchar 填充空位
    str.encode(encoding=’utf-8’)返回原字符串编码为字节串对象的版本
    str.format(* args, ** kwargs) 执行字符串格式化操作
    str.isalnum() 均为字母或数字返回True
    str.isdecimal() 均为十进制数字返回True
    str.isdigit()均为数字返回True
    str.title() 返回原字符串的标题版本

元组

元组是不可变类型，用()创建

  • 元组操作
    tuple()将任意序列或迭代器转换为元组
    元组可以进行加法或乘法，返回一个新对象
    元组可以进行拆包解包操作：a,b=b,a
    a,b, * rest = tuple可以将多余的元素去除，rest是一个列表
    * 是一个解包符号
    可以使用count方法统计某个值出现的频率

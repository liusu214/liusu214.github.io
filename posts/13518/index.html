ffmpeg使用

十二月 20, 2019
  • 软件

1.常用命令

1.1 主要参数

-i 设定输入流
-f 设定输出格式
-ss 开始时间

1.2 视频参数

-r 设定帧速率，默认为25
-s 设定画面的宽与高
-aspect 设定画面的比例
-vn 不处理视频
-vcodec 设定视频编解码器，未设定时则使用与输入流相同的编解码器

1.3 音频参数

-ac 设定声音的Channel数
-acodec 设定声音解编码器，未设定时则使用与输入流相同的编解码器
-an 不处理音频

2. 实际功能

  • 2.1 视频格式转换

  • 提取音频

  • 提取视频

  • 视频剪切

-ss设置从视频的哪个时间点开始截取，上文从视频的第2s开始截取。
-t表示截取多长的时间，上文截取的视频共3s。
-vcodec copy表示使用跟原视频一样的视频编解码器。
-acodec copy表示使用跟原视频一样的音频编解码器。
-i 表示源视频文件
-y 表示如果输出文件已存在则覆盖。
-to 截到视频的哪个时间点结束
-c:v 和 -c:a分别指定视频和音频的编码格式。
-c:v copy -c:a copy标示视频与音频的编码不发生改变，而是直接复制，这样会大大提升速度。

  • 码率控制
    码率(Kbps) = 文件大小(bit) / 时长(s)

-b:v 控制平均码率。建议加上-bufsize用于设置码率控制缓冲器的大小
-minrate 码率波动的阈值

  • 修改视频的分辨率

  • 添加logo

  • 抓取视频帧

-r 表示每一秒几帧
-q:v 表示存储jpeg的图像质量，一般2是高质量
-f 指定文件格式

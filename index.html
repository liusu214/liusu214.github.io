列表、字典、字符串操作

发表于 2019-09-25

列表

列表是可变类型，用[]或list来定义

  • 列表操作
    list[i:j:k] 列表从0 <= n < (j-i)/k 的索引号 x = i + n* k 的切片
  • list[i:-1]不包括最后一个元素，其他同理*
    list.index(a) a在列表中首次出现的索引
    len(list)、max(list)、min(list) 列表的长度、最大值、最小值
    del list 删除列表
    list[index] = x 将列表index项替换为i,找不到会报错
    list.append()在列表末尾添加元素
    list.insert(index,a)在特定位置插入元素，耗费计算量大，因为对后续元素的引用必须在内部迁移，以便为新元素提供空间。
  • 当index是-1时，会插入到最后一个值前*
    list.pop(index)删除指定位置的元素，并返回该元素。不填默认删除最后一个，找不到会报错
    list.remove(值)删除第一个值，找不到会报错
    list.extend()可以添加多个元素
    list.sort()可以对列表进行排序，可以添加key指定排序规则，可以设置reverse=True进行反向排序(默认为False)
    list.reverse()反转列表
    ‘-‘.join(list)，以-为串联符号将列表内元素串联成字符串
  • 序列函数
    enumerate函数可以返回(index,value)的元组序列
    sorted函数，跟sort类似
    zip函数,可以将多个列表、元组或其他序列成对组合成一个元组列表

zip可以被用来解压序列，也可以转换行列表为列列表

reversed(list) 反转序列，返回一个迭代器

字典

字典是可变类型，用dict()来定义，以键值对的形式表示

  • 字符串操作
    dict[‘key’]访问与键对应的值，没有会报错
    dict[key] = value，若有则覆盖，没有则新建
    del dict 删除字典 dict.pop[key] 删除键值对，没有会报错 dict.popitem()从字典中移除最后一个并返回一个键值对
    dict.keys() dict.value() 输出键和值
    dict.update(dict1) 将一个字典和另一个融合
    dict.get(key) 返回值，*不会报错 *

dict.setdefault(键,数据) 键值对不存在添加键值对，存在则不做处理
dict.setdefault(key, []).append(value) 用于for循环中没有key就新建列表，有则添加到列表中
dict.items()获取所有键和值，用两个遍历变量
dict.values() 返回由字典值组成的新视图

  • 字典方法
    len(dict) 返回字典的项数
    iter(dict) 返回以字典的键为元素的迭代器

字符串

字符串为不可变类型，用str()转化为字符串

  • 字符串操作
    string.isalpha()
    所有都是字母返回True
    string.isdecimal()
    所有都是数字返回True
    string.islower()
    都是小写字母返回True
    string.isupper()
    都是大写字母返回True
    string.startswith(“a”)
    检查字符串是否是以 a开头，是则返回 True
    string.endswith(“a”)
    检查字符串是否是以 a结束，是则返回 True
    string.find(str, 0,5) 查找数据第一次出现的索引值，找不到返回-1，也可以指定范围查找
    string.rfind(str) 反向查找
    string.count(数据) 类似find()，找不到会报错
    string.index(str) 类似rfind()，找不到会报错
    string.replace(‘旧内容’,’新内容’, ‘替换次数’) 替换字符串中的内容，实际上是新建一个字符串
    string.split(str, maxsplit=-1) 以str为分隔符拆分成列表并返回，给出maxsplit则进行maxsplit此拆分
    string.lower()转为小写
    string.upper()转为大写
    string.lstrip() 把左边的空格去除
    string.rstrip()把右边的空格去除
    string.strip() 把左右两边的空格去除
    str.capitalize() 返回原字符串的副本，首个字符大写，其他小写
    str.casefold() 返回原字符串消除大小写的副本
    str.center(width,fillchar) 返回长度为width的字符串，原字符串在中间用fillchar填充两边的空位， 若width小于等于len(str)则返回原字符串的副本
    str.rjust(width[, fillchar])
    返回长度为 width 的字符串，原字符串在其中靠右对齐,
    使用指定的 fillchar 填充空位
    str.encode(encoding=’utf-8’)返回原字符串编码为字节串对象的版本
    str.format(* args, ** kwargs) 执行字符串格式化操作
    str.isalnum() 均为字母或数字返回True
    str.isdecimal() 均为十进制数字返回True
    str.isdigit()均为数字返回True
    str.title() 返回原字符串的标题版本

统计字符串中每一个字符出现的次数

元组

元组是不可变类型，用()创建

  • 元组操作
    tuple()将任意序列或迭代器转换为元组
    元组可以进行加法或乘法，返回一个新对象
    元组可以进行拆包解包操作：a,b=b,a
    a,b, * rest = tuple可以将多余的元素去除，rest是一个列表
    * 是一个解包符号
    可以使用count方法统计某个值出现的频率

python内置模块

1.os模块

os.getcwd()：查看当前工作目录，返回工作目录
os.chdir(path)：将当前工作目录更改为指定的路径，path可以是相对路径或绝对路径
os.getpid()：返回当前进程ID
os.getppid():返回父进程ID
os.mkdir(path,mode):path目录，mode-权限数字模式
os.remove(path):删除路径为path的文件
os.removedirs(path):递归删除目录
os.listdir(path):返回path指定的文件夹包含的文件列表
os.stat(path):获取文件或目录信息

  • os.path
    os.path.abspath(path):返回绝对路径
    os.path.dirname(path):返回上一级目录
    os.path.getsize(path):返回路径的大小(字节单位)
    os.path.split(path):将路径拆分成一对，返回路径和文件名
  • os.path.splitext(path):将文件名和扩展名分开*

2.sys模块

sys.argv[] 获取运行python文件时命令行参数，以列表形式存储 sys.grgv[0]代表当前文件名
sys.exit():退出程序
sys.getdefaultencoding():获取系统当前编码
sys.setdefaultencoding():设置系统默认编码
sys.path: 获取指定模块搜索路径的字符串集合

3.re模块

符号 解释 示例 说明
. 匹配任意字符 b.t 可以匹配bat / but / b#t / b1t等
\w 匹配字母/数字/下划线 b\wt 可以匹配bat / b1t / b_t等
但不能匹配b#t
\s 匹配空白字符（包括\r、\n、\t等） love\syou 可以匹配love you
\d 匹配数字 \d\d 可以匹配01 / 23 / 99等
\b 匹配单词的边界 \bThe\b
^ 匹配字符串的开始 ^The 可以匹配The开头的字符串
$ 匹配字符串的结束 .exe$ 可以匹配.exe结尾的字符串
\W 匹配非字母/数字/下划线 b\Wt 可以匹配b#t / b@t等
但不能匹配but / b1t / b_t等
\S 匹配非空白字符 love\Syou 可以匹配love#you等
但不能匹配love you
\D 匹配非数字 \d\D 可以匹配9a / 3# / 0F等
\B 匹配非单词边界 \Bio\B
[] 匹配来自字符集的任意单一字符 [aeiou] 可以匹配任一元音字母字符
[^] 匹配不在字符集中的任意单一字符 [^aeiou] 可以匹配任一非元音字母字符
* 匹配0次或多次 \w*
+ 匹配1次或多次 \w+
? 匹配0次或1次 \w?
{N} 匹配N次 \w{3}
{M,} 匹配至少M次 \w{3,}
{M,N} 匹配至少M次至多N次 \w{3,6}
| 分支 foo|bar 可以匹配foo或者bar
(?#) 注释
(exp) 匹配exp并捕获到自动命名的组中
(?P<name>exp) 匹配exp并捕获到名为name的组中
(?:exp) 匹配exp但是不捕获匹配的文本
(?P=exp) 匹配exp前面的位置 \b\w+(?=ing) 可以匹配I’m dancing中的danc
(?<=exp) 匹配exp后面的位置 (?<=\bdanc)\w+\b 可以匹配I love dancing and reading中的第一个ing
(?!exp) 匹配后面不是exp的位置
(?<!exp) 匹配前面不是exp的位置
*? 重复任意次，但尽可能少重复 a.*b
a.*?b		 将正则表达式应用于aabab，前者会匹配整个字符串aabab，后者会匹配aab和ab两个字符串
+? 重复1次或多次，但尽可能少重复
?? 重复0次或1次，但尽可能少重复
{M,N}? 重复M到N次，但尽可能少重复
{M,}? 重复M次以上，但尽可能少重复

4.datetime模块

class datetime.date 设置时间(年月日)

  • date.replace(year=self.year …) 返回一个设置日期一样的值
  • date.weekday()返回一个整数代表星期几，星期一为0，星期天为6*
    date.isoweekday()同上星期一为1，星期天为7
    date.isocalendar()返回年、周、星期的元组
    date.isoformat() 返回YYYY-MM-DD
    date.today()返回今天日期

class datetime.time 设置时间(时分秒时区)
class datetime.datetime 上面两个集合

  • datetime.today()返回当前本地时间
    datetime.utcnow()返回标准时间
    datetime.replace()返回一个设置日期一样的值
    datetime.astimezone(tz=None)返回一个新的tzinfo属性tz的datetime对象
    datetime.weekday() datetime.isoweekday()等同date

5.time模块

time.sleep(secs)，暂停执行xx秒
time.strftime()转换一个元组为时间字符串

6.hashlib模块

hashlib模块实现了md5，sha1，sha512等算法，通过hashlib.algorithms_available
用法

hashlib.new(name[, data])
通用构造函数，name参数表示要使用的hash算法的名称，可以是上述列出的所有hash算法以及OpenSSL库支持的其他算法
hash.digest_size
hash对象的字节长度
hash.block_sizehash
对象的内部块大小
hash.namehash 对象的名称
hash.update(data)传递类字节参数(通常是bytes)更新hash对象

7. logging模块

日志等级：DEBUG、INFO、WARNING、ERROR、CRITICAL
开发时应使用INFO或DEBUG级别，部署时使用WARNING或ERROR级别
